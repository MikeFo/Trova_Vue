# Trova Color Palette

Based on the Trova logo (#34A853)

## Primary Colors

### Primary Green (Logo Color)
- **Base**: `#34A853` (RGB: 52, 168, 83)
- **Light**: `#48b164` (for hover states, tints)
- **Dark**: `#2e9449` (for shade states, pressed buttons)
- **Usage**: Primary buttons, links, brand elements, CTAs

### Secondary Teal
- **Base**: `#1DB98A` (RGB: 29, 185, 138)
- **Light**: `#34c096`
- **Dark**: `#1aa279`
- **Usage**: Secondary actions, accents, complementary elements

### Accent Coral
- **Base**: `#FF6B6B` (RGB: 255, 107, 107)
- **Light**: `#ff7a7a`
- **Dark**: `#e05e5e`
- **Usage**: Highlights, alerts, important notifications

## Semantic Colors

### Success
- **Base**: `#10b981` (Green variant)
- **Usage**: Success messages, confirmations

### Warning
- **Base**: `#f59e0b` (Amber)
- **Usage**: Warnings, cautions

### Danger/Error
- **Base**: `#ef4444` (Red)
- **Usage**: Errors, destructive actions

## Neutral Colors

### Dark
- **Base**: `#1a1a1a`
- **Usage**: Primary text, headings

### Medium
- **Base**: `#6b7280`
- **Usage**: Secondary text, labels

### Light
- **Base**: `#f9fafb`
- **Usage**: Backgrounds, cards

### Border
- **Base**: `#e5e7eb`
- **Usage**: Dividers, input borders

## Gradient Combinations

### Primary Gradient
```css
background: linear-gradient(135deg, #34A853 0%, #1DB98A 100%);
```

### Alternative Gradient
```css
background: linear-gradient(135deg, #2e9449 0%, #34A853 100%);
```

## Usage Guidelines

1. **Primary Green** should be used for:
   - Primary call-to-action buttons
   - Brand elements (logo backgrounds, headers)
   - Active states
   - Links and navigation highlights

2. **Secondary Teal** should be used for:
   - Secondary buttons
   - Complementary accents
   - Progress indicators
   - Secondary actions

3. **Accent Coral** should be used sparingly for:
   - Important highlights
   - Special notifications
   - Feature badges

4. **Maintain contrast** - Ensure text on colored backgrounds meets WCAG AA standards (4.5:1 ratio)
